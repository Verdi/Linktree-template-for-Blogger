# Linktree-template-for-Blogger
Simple, free website
## How to Use
1. Log into [blogger.com](https://blogger.com) with your Google account.
2. Create a blog.
3. Click on Theme in the sidebar, and then choose "edit HTML" from the Customize dropdown.
